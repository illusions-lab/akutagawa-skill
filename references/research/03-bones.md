### ジャンル別段落呼吸

| ジャンル | 1 段落の文数目安 | 1 段落の字数目安 |
|---------|------------:|------------:|
| 古典物・歴史物（地の文） | 3〜6 文 | 200〜500 字 |
| キリシタン物 | 4〜8 文 | 250〜700 字 |
| 童話 | 2〜4 文 | 80〜250 字 |
| 自伝・私小説 | 2〜5 文 | 100〜300 字 |
| 晩年告白物（『歯車』『或阿呆の一生』） | 1〜3 文 | 50〜200 字（番号付き断章） |
| 警句集（『侏儒の言葉』） | 1〜2 文 | 20〜100 字（独立警句） |
| 対話文 | 1 文 | 改行ごとに 1 段落 |

### 改行・改段の癖

- **改段は律儀に切る**：場面・心理・話者の転換ごとに段落を改める。「改段の経済性」が芥川の特徴
- **一行空けは基本使わない**：散文の中での「白」の表現は控える
  - **例外**：晩年の番号付き断章（『或阿呆の一生』51 章、『歯車』6 章）。ここでは章番号と章題の前後で空白行を取る
- **冒頭の段落構成**（古典物）：時刻・季節・天候 → 場所・場面 → 人物導入 → 状況・背景 → 心理・行動 へと **段階的に絞り込む**

### 緩急パターン

芥川的呼吸の核は **「短い断定文 → 中位の修飾文 → 短い断定文」**の三拍子である。これに **超長文（独白または対句）** が時に挟まる四拍子の変奏もある。

**運用例（仮構）**：
```
1 文目（短）：男はそこに立つてゐた。
2 文目（中）：辺りには、夕日の傾いた荒野の他に、何も見えなかつた。
3 文目（短〜中）：男は、ふと、空を仰いだ。
4 文目（長）：すると、薄墨色の雲の中に、半透明の——いや、おそらくは黄金の影が、彼の網膜を射抜くやうに浮び上がつた。
5 文目（短）：然し、それも一瞬の事である。
```

これが「刀の切先」の呼吸である。短文で切り、中文で巻き、長文で切先を伸ばし、また短文で切る。

### 段落構造の運用ルール

- 古典物・歴史物：段落は中央値 200〜400 字、文数 3〜5
- 童話：段落は 80〜200 字、ひらがな比率を上げる
- 晩年告白：段落は 50〜200 字、番号付き断章は許容
- 警句集：1 段落 1 警句、20〜100 字
- 一段落に超長文（中央値文長 25 字 × 2.5 = 60 字以上）は **0〜1 本まで**
- 連続して 3 段落以上、長段落（500 字超）を重ねるのは古典物・歴史物の鬼気迫る場面のみ

## L8 対話処理

### 会話比率

作品ごとに大きく異なる：

| 作品タイプ | 会話比率の傾向 |
|----------|----------|
| 古典物（『羅生門』『鼻』『地獄変』） | 中（30〜40%）。地の文と対等 |
| **多視点物（『藪の中』）** | **極めて高（60〜80%）**。各話者の独白が主体 |
| 童話（『蜘蛛の糸』『杜子春』） | 中（20〜35%）。三人称地の文が主 |
| キリシタン物 | 低〜中（15〜30%）。古文献を訳した如き地の文中心 |
| 晩年告白物（『歯車』『或阿呆の一生』） | 低（10〜25%）。一人称独白が支配 |
| 警句集 | ほぼ 0%（地の文のみ） |

### 「」の扱い

- **基本形**：会話ごとに改行、「」で囲む近代小説標準
- **発話タグ**：「〜」と○○は言つた。／○○は「〜」と言つた。両方を併用
- **発話タグ省略**：話者が文脈で明確な場合は省略（連続対話の中盤）
- **「」の中の「」**：『』を内側に使う標準
- **長い独白**：1 段落の中で 「」を閉じずに、行を変えて続けることも（『藪の中』の各証言の長尺）

### 方言・擬古語の使い分け

- **方言**：原則使わない。芥川の登場人物は階層・職業・性差で語彙を変えるが、地域方言（東北弁・関西弁）は基本登場しない
- **擬古語**（古典物の時代語）：『羅生門』『地獄変』の中世風話法、『藪の中』の武士の文語、『神々の微笑』のキリシタン宣教師語など——**時代に合わせた擬古的セリフ**を構成する。実際の中世文献を引くのではなく、**読みやすさを保ちつつ古めかしさを纏う**
- **西洋人**：宣教師オルガンチノ（『神々の微笑』）など、「君」「貴下」「我々」を多用するやや片言の日本語、または翻訳調のセリフ
- **童話の動物**：『犬と笛』『白』など——人語を話すが、敬語を交えた **童心的な丁寧体**

### 話者差異の構成法

身分・教養差を語彙レベルで描き分ける：

| 話者類型 | 語彙・文末の特徴 |
|--------|------------|
| 武士・上級武家 | 「拙者」「左様」「〜であらう」「〜申さう」（文語助動詞） |
| 僧・法師 | 「拙僧」「〜と申す」「〜にて候」（仏教漢語、候文） |
| 商人・町人 | 「手前」「〜でござります」「〜だなえ」（江戸町人語） |
| 盗賊・無頼 | 「俺」「〜だぜ」「〜じやねえか」（俗語混じり） |
| 女・夫人 | 「わたし」「あたくし」「〜ですわ」「〜かしら」（敬体・終助詞） |
| 西洋人（宣教師等） | 「我々」「〜であります」「〜と云ふものは〜」（翻訳調） |
| 子供 | 「ぼく」「あたし」「〜だい」「〜なの」（短文・終助詞「の」「な」） |
| 河童（『河童』） | 人間語に擬しつつ「カラカラ」「ケエル」など擬音化された名詞を混ぜる |

### 対話における運用ルール

- 古典物・歴史物では話者の **身分を語彙で示す**（「拙者」「拙僧」「手前」など）
- 武士の文語、僧の候文、町人の俗語は **過剰に擬古しない**——読者に古文献ではなく短篇小説と感じさせる節度
- 連続対話の応酬では、発話タグを段階的に省略し、テンポを上げる
- 西洋人の日本語は **翻訳調**（漢語使用率の高さ、文の論理的構成、感情語の控えめ）
- 方言は使わない（古典物の時代語で代替）

## L9 物語構造

芥川は「短篇の大家」と呼ばれた。短篇 150 編以上を遺し、長篇は『邪宗門』『路上』ともに未完。**短篇構造の精度**こそが彼の生涯の課題であり、芸術性の源泉だった。

### 短篇結末型の分類

| 結末型 | 代表作 | 特徴 |
|------|------|------|
| **冷ややかな転回型** | 羅生門・鼻・芋粥 | クライマックスの直後、語り手が **皮肉的に距離を取る一文**で締める |
| **悲劇的崩壊型** | 地獄変 | 主人公が完成と引き換えに自己を破壊、語り手はその崩壊を冷然と記述 |
| **多視点宙吊り型** | 藪の中・報恩記 | 真実が決定不能のまま閉じる、読者に判断を委ねる |
| **救済否定型** | 蜘蛛の糸 | 一旦提示された救済が、人間のエゴで自壊する |
| **覚醒型** | 杜子春 | 仙人になる夢を諦めて人間に戻る、現代的な目覚め |
| **円環回帰型** | 戯作三昧 | 冒頭の状態に主人公が戻るが、内面で何かが変わつてゐる |
| **断章連鎖・崩壊型** | 歯車・或阿呆の一生 | 物語構造そのものが断片化、最後は「もう書きつづける力がない」型 |
| **黙示録型** | 河童・西方の人 | 寓話的・神話的構造で文明批評を展開 |

### 起承転結／三幕の構成

**典型的な起承転結（古典物）**：

| 幕 | 役割 | 例（『羅生門』） |
|----|------|------------|
| 起 | 状況設定・人物導入 | 雨やみを待つ下人、京の荒廃 |
| 承 | 状況の深化・葛藤の前提 | 楼上に登る、死人の姿 |
| 転 | 事件・決定的場面 | 老婆との対峙、髪を抜く老婆、下人の怒り |
| 結 | 結果と語り手の距離化 | 「下人の行方は、誰も知らない」 |

**幕間の比率**：起 25% / 承 25% / 転 35% / 結 15% が芥川の典型。**転と結が短く切り上げられる**のが特徴——転の頂点をだらだら引き伸ばさず、結ぐで一気に冷却する。

### 場面転換

- **明示的**：「翌日」「数日後」「やがて」「果して」など時間副詞で時空の移動を律儀に告げる
- **章分け／節分け**：長めの短篇では「一」「二」「三」と漢数字で節を切る（『地獄変』『戯作三昧』『藪の中』）
- **時間軸の操作**：『藪の中』のような同一事件の多視点反復、『歯車』の幻覚と現実の交錯——晩期に複雑化

### 冒頭の典型構造

古典物の冒頭は以下の **3〜4 段ステップ**で世界に入る：

1. **時刻・季節・天候**（「或日の暮方の事である」「或寒い日の朝」）
2. **場所**（「羅生門の下」「池の上」「暗の中」）
3. **人物導入**（「一人の下人が」「老婆が」「内供は」）
4. **状況・背景**の補足（「雨やみを待つてゐた」「京には〜の災ひが続き〜」）

**運用例**：
- 『羅生門』：「或日の暮方の事である。一人の下人が、羅生門の下で雨やみを待つてゐた。」
- 『鼻』：「禅智内供の鼻と云へば、池の尾で知らない者はない。」
- 『地獄変』：「堀川の大殿様の様な方は、これまでは固より、後の世にもまた、二人とはいらつしやいますまい。」（語り手が登場するパターン）

### 結句の典型構造

**冷却の結句**は芥川の代名詞。クライマックスの熱を一気に冷ます：

- 『羅生門』：「下人の行方は、誰も知らない。」
- 『鼻』：「内供は、せめてもの恨みに、頻りに鼻のかいなを掻いた。」（短く落とす）
- 『芋粥』：「五位の前には、ただ、冷えた芋粥の鍋が残つてゐた。」（仮構）
- 『地獄変』：「良秀は……（中略）……自ら己が首を縊つて死んだ。」（簡素な事実報告）
- 『蜘蛛の糸』：「やはり、無慈悲な人間の心が、罰を受けたのである。」（教訓化を装いつつ皮肉）

**運用ルール**：
- 結句は **短く**（冒頭文の半分以下が目安）
- 結句で **熱を冷ます**——感情の頂点を結句に置かない
- 「〜であつた」「〜のだ」「〜である」で締める
- 「教訓を直接書く」のは芥川的にはギリギリ許容（『蜘蛛の糸』『杜子春』）。ただし **皮肉の含み**を残すこと

### クライマックス位置

- **起承転結の「転」の中盤〜終盤**にクライマックスを置き、結句直前で頂点を作る
- クライマックスの直後、**1〜2 文で距離を取る**（「然し、それは……」「下人は〜と思つた。然し〜」）
- 頂点を **長尺の独白で引き伸ばさない**——芥川の節度

### 円環構造

**冒頭の風景描写が結末で反転して回帰する円環**は芥川の隠れた癖：

- 『戯作三昧』：馬琴の家から始まり、馬琴の家で終わる（しかし内面は変化している）
- 『地獄変』：絵師の存在から始まり、絵師の死で終わる
- 『歯車』：歯車の幻視から始まり、歯車の幻視で終わる

これは **発端と結末の対応**による意味の累積化を生む。

### 物語構造の運用ルール

- **依頼が短篇の生成・続き予測の場合**：起承転結を意識し、転の頂点を結句で冷却する
- **依頼が長篇の場合**：芥川は **長篇を書けなかった**——長篇要請には「短篇連作」あるいは「断章型」を提案する（晩年の『歯車』『或阿呆の一生』形式）
- **多視点の依頼**：各話者の声を独立させ、地の文に総括を入れない（『藪の中』型）
- **童話の依頼**：救済の結末でも **皮肉の含み**を残す（『蜘蛛の糸』のカンダタ、『杜子春』の覚醒）
- **警句の依頼**：冒頭は断定、続いて自己取消（「——いや、そんなはずもあるまい」）で揺らぎを残す
- **長い長尺の独白を要求された場合**：晩年告白モード（『歯車』）で対応するが、それでも 1 段落 200 字超の独白は連続させない
