# 02 眼（Eye）— L4〜L6

> **出典**：青空文庫 374 篇のコーパス精読、および主要代表作（羅生門 1915 / 鼻 1916 / 戯作三昧 1917 / 地獄変 1918 / 蜘蛛の糸 1918 / 杜子春 1920 / 藪の中 1922 / 報恩記 1922 / 神々の微笑 1922 / 歯車 1927 / 河童 1927 / 或阿呆の一生 1927 / 侏儒の言葉 1923-27）の語り手解析。
> **方針**：L4-L6 は数値より作品単位の質的分析を主とする。L1-L3 で確定した数値（自称代名詞 46% が「僕」、文末「〜た」27.5%、漢字比率 31.2%）と矛盾しない範囲で、視点・距離・五感の運用ルールを抽出する。

---

## L4 視点・時制

芥川の語り手は **作品ジャンルによって明確に切り替わる**。これが最も大きな運用上の判断点となる。

### ジャンル別視点プロファイル

| ジャンル | 代表作 | 人称・視点 | 語り手距離 | 時制 |
|---------|------|---------|---------|------|
| **古典物・歴史物** | 羅生門・鼻・芋粥・地獄変・戯作三昧 | 三人称客観 | 冷ややかな観察者・透明 | 過去形「〜た／〜であつた」 |
| **古典物・多視点** | 藪の中・報恩記 | 複数の一人称告白の連鎖（自由間接話法も） | 各話者の主観に貼り付き、結末で宙吊り | 各告白は過去形 |
| **キリシタン物・宗教物** | 奉教人の死・神々の微笑・西方の人 | 三人称（語り手の擬古的距離）または書記者風 | 古文献を訳した如き距離 | 過去形・歴史的現在 |
| **童話・赤い鳥系** | 蜘蛛の糸・杜子春・犬と笛・白 | 三人称（語り聞かせ調） | 子供読者への配慮、優しさ | 過去形 |
| **開化物・近代物** | 開化の良人・舞踏会・蜜柑 | 三人称限定 or 一人称回想 | 中等の親密 | 過去形 |
| **自伝的・私小説** | 大導寺信輔の半生・点鬼簿・玄鶴山房 | 三人称ながら主人公に密着 | 自己との距離が緩む | 過去形・現在形混在 |
| **晩年告白物** | 歯車・或阿呆の一生・河童 | 一人称「僕」（『河童』は別世界の語り手） | 主観の崩壊、内的時間 | 現在形・過去形交錯 |
| **警句集・随筆** | 侏儒の言葉・文芸的な、余りに文芸的な | 一人称「僕」「我々」 | 著者そのもの | 現在形・断定 |

### 古典物・歴史物の語り手（最も典型的な芥川）

**特徴**：
- **完全な三人称客観**。「下人」「老婆」「良秀」のように、固有名詞または役割名で人物を指す
- 語り手は透明だが、結末で **皮肉的に距離を取る**（「下人の行方は、誰も知らない」型）
- **冒頭の風景描写**で世界の温度を一瞬で決定する：「或日の暮方の事である。一人の下人が、羅生門の下で雨やみを待つてゐた」
- 心理描写は **観察された事実**として書かれ、「内面の独白」を装わない。「下人は……と思つた」「老婆は……と感じた」のような **間接的心理語り**

**例（『羅生門』冒頭の構造分解）**：
```
1. 時刻指定（暮方）→ 季節指定（雨）→ 場所指定（羅生門）
2. 人物導入（下人、一人）→ 行動（雨やみを待つ）
3. 状況説明（饑饉、地震、火事、辻風）
4. 内面（途方に暮れる）— 直接的でなく、状況の必然として
```

これは「映画的カメラワーク」と評される構造である。

### 多視点の運用（『藪の中』型）

**藪の中**：同一事件（武士の死）について、7 人の語り手（樵・旅法師・放免・老婆・盗人・女・霊媒を介した武士）が告白する。各証言は **互いに矛盾し、真実は最後まで宙吊り**にされる。

**運用ルール**：
- 各話者の声は **語彙・文末・敬体／常体で明確に区別**する（武士の堂々たる文語、盗人の俗語混じり、女の哀切な敬体）
- 語り手（芥川）は **一切の総括を放棄**する。読者に判断を委ねる
- これは多視点を装って真実を再構築するミステリーではなく、**真実そのものの不可知性**を構造化した小説

### 晩年告白物の一人称（『歯車』『或阿呆の一生』）

**特徴**：
- 一人称「僕」基調
- **時制が崩れる**：過去の回想と現在の幻覚が同列に並ぶ。「僕は、その時、半透明の歯車を見た。歯車は今もまだ、廻つてゐる」型の時制混淆
- **断章化**：『或阿呆の一生』は 51 章、各章 1〜2 段落の番号付き断片
- 観察距離を保とうとする芥川と、距離が崩壊しつつある芥川の **相剋**そのものが文体になる

**運用ルール（晩年モード）**：
- 一人称「僕」、現在形と過去形を交錯
- 段落・章を断片化（番号付き断章）
- 心理描写は直接的だが、**観察的であり続ける**（「僕は怖かつた」ではなく「僕は何かに怯えてゐた——それが何であるかは、僕にも判らなかつた」）
- 自己の崩壊を語っても、語ることそのものは冷静に行う

### 童話の語り手（『蜘蛛の糸』『杜子春』）

**特徴**：
- 三人称、語り聞かせ調（「お釈迦様が極楽の蓮池のふちを、独りでぶらぶら御歩きになつていらつしやいました」）
- 敬語・敬体を時に混ぜる
- 子供読者への配慮で漢字を控え、ひらがな比率を上げる
- ただし **教訓は冷ややか**——『蜘蛛の糸』は救済を否定し、『杜子春』は仙人になる夢を放棄させる

### 自由間接話法

限定的だが、『藪の中』のような複数視点劇、『地獄変』の良秀の狂気、『歯車』の主人公の内的崩壊では **自由間接話法**が機能する。直接話法（「」）と地の文の境界を曖昧にし、**人物の意識が地の文を侵食**する瞬間を作る。

## L5 五感プロファイル

### 偏愛感覚の序列

**視覚 ≫ 聴覚 ≧ 触覚 > 嗅覚 ≧ 味覚**

芥川は徹底的に **視覚優位の作家**である。光・色彩・形状の描写は緻密、聴覚・触覚は次点、嗅覚・味覚は限定的。

### 視覚

**核心モチーフ**：
- **火**（地獄変の炎、河童の焼き場、歯車の発光）
- **血**（羅生門の死人、地獄変の業火、藪の中の刀傷）
- **金**（黄金、金襴、金粉、貨幣）
- **闇**（夜、雨、煙、靄）
- **光と闇の対比**（薄明、夕闇、暮方、月光）
- **色彩**：赤（火・血・着物）、黒（闇・髪）、白（雪・月・蓮華）、金（黄金・金粉）が主軸。中間色（緑・青）は控えめ

**運用例**：
- 『地獄変』：「立て並べた黄金の燭台の灯が、白々と〜」「絵筆の動くが如くに、もう紅蓮の炎は柱を巻いて〜」
- 『羅生門』：「夕闇は次第に空を低くして〜」「暗の中から、烏の声が一声」
- 『歯車』：「半透明の歯車が、僕の視野の隅で廻つてゐる」（閃輝暗点の描写）

### 聴覚

**頻出**：
- 風（吹く・鳴る）、雨（降る・打つ）、雷、虫の音、烏・鴉の声
- 楽器：琵琶・笛・三味線・ピアノ（西洋風物として）
- 鐘（寺院・教会）、足音、衣擦れの音

聴覚描写は **視覚を補強する役**として配置されることが多い。「闇の中から〜の声」型は芥川の常套。

### 触覚

**頻出**：
- 寒暖（冷たい・熱い）、湿度（湿つた・乾いた）、肌触り（柔らかい・荒れた）
- 風の触感、雨の感触、毛布の重さ
- 晩年の『歯車』では病的な触覚（皮膚の冷感、強迫的な手の感覚）

### 嗅覚

**頻出**：
- 煙（煙草、火事、線香）、酒の匂い、香（線香・名香）、血の匂い（限定的）
- 西洋風物：コオヒイ、葉巻、香水（限定的）

嗅覚は描写の比重が低い。芥川は喫煙者（『敷島』を一日 180 本）だが、煙草の匂いは作品に過剰に持ち込まない。

### 味覚

**最も希薄**。食事の場面そのものが少ない（『芋粥』の山盛り芋粥が例外的に味覚を主題化）。

### 季節・光・気温の描写密度

**冒頭の天候・時刻描写**で世界の温度を瞬時に決める癖が顕著。

- 「或日の暮方の事である」（『羅生門』）
- 「ある寒い日の暮方である」（『鼻』）
- 「或曇つた冬の日暮である」（よくある型）
- 「天保年間のある雪の朝」（『戯作三昧』）

**運用ルール**：
- 冒頭文に **時刻・季節・天候のいずれか**を 1 つ以上入れる（ほぼ義務）
- 中盤での天候の変化は **物語の転回**と連動させる（『地獄変』の絵師の狂気と火の燃え盛り）
- 結末で天候を回帰させ、**円環的構造**を作る場合もある

## L6 読者距離

### 二人称の使用

- **小説本文**：稀。三人称客観を貫くため、読者を「諸君」「君」と呼ぶことはほぼない
- **書簡・随筆**：限定的に使用。「君」「貴下」「貴兄」など宛先による
- **警句集**（『侏儒の言葉』）：「諸君」「我々」「君」が頻出。**警句集においてのみ二人称・複数人称が解禁**される

### 呼びかけ構文

**極めて寡少**。読者に直接呼びかける文（「読者諸君に問う」型）は、警句集と評論を除き、ほぼ出現しない。

**例外**：
- 『侏儒の言葉』：「諸君は経験から物を学ぶと云ふ。然し〜」
- 『文芸的な、余りに文芸的な』（評論）：「我々は谷崎君の意見に〜」

### 語り手の存在感

| 作品ジャンル | 語り手の存在感 |
|------------|------------|
| 古典物・歴史物 | **透明**（語り手は黒衣として消える） |
| 童話 | 軽く存在（語り聞かせ調） |
| キリシタン物 | 訳者・書記者風（古文献の翻訳的距離） |
| 自伝的・私小説 | 中等（主人公にほぼ密着） |
| 晩年告白物 | **強く押し出す**（一人称「僕」が思考を露出） |
| 警句集 | **最も強い**（直接「我々」「諸君」と語る） |

**読解**：芥川の語り手は、**ジャンルによってグラデーション**を持つ。古典物では透明、晩年告白物では露出、警句集では直接読者と対峙する。skill ロード時の応答ジャンルに応じてこのグラデーションを再現する必要がある。

## 運用上の判断ツリー

依頼内容から、どの視点モードを取るかを以下で判断：

```
依頼ジャンルは何か？
├─ 古典物・歴史物（武士・僧・盗賊・絵師・宮廷など）
│   → 三人称客観、過去形、語り手透明、冒頭天候描写、結末で距離化
├─ 多視点・矛盾劇（複数の証言・告白）
│   → 各話者の声を独立させる、地の文に総括を入れない
├─ キリシタン物・古典宗教物
│   → 三人称、擬古的距離、訳者風、漢字密度を上げる
├─ 童話・寓話
│   → 三人称、語り聞かせ調、敬体混入、ひらがな比率を上げる
├─ 開化物・近代物
│   → 三人称または一人称回想、明治大正の風物を入れる
├─ 自伝的告白・心境小説
│   → 三人称密着 or 一人称「僕」、過去と現在の交錯
├─ 晩年告白物・幻覚小説
│   → 一人称「僕」、現在形と過去形の交錯、断章化、自己崩壊を観察的に語る
└─ 警句・アフォリズム
    → 一人称「僕」または「我々」、断定、対句、自己取消で揺らぎを残す
```

**何でも応答してしまう前に、ジャンルを判定する**——これが芥川的応答の出発点である。
